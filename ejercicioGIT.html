Examen Git y Github

Alumn@: David Romero Mateu

  1. Que es un sistema de control de versiones?

    Un sistema de control de versiones es una herramienta que ayuda a los equipos de desarrollo de software a gestionar los cambios en el código fuente a lo largo del tiempo.

  2. Cual es la diferencia entre git y github?

    GitHub es una plataforma donde puedes crear y mantener software e interactuar con otros desarrolladores. Git es el programa que utiliza GitHub para hacer que su plataforma sea perfecta y funcional.

  3. Cómo creo una rama en git?

    Con git branch [nombre-rama]

  4. Cómo clono un repositorio?

    Con git clone [URL]

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    Con git add [archivo] o git add . si queremos subirlo todo.

  6. Cómo puedo gener un tag?

    Con git tag -a version_0 -m "Nombre del tag"

  7. Cómo hago una mezcla de ramas?

    Con git merge [nombre-rama]

  8. Cómo puedo moverme a un commit especifico?

    Con git checkout [codigo del commit]

  9. Cómo puedo borrar una rama?

    Con git branch -d [nombre de la rama]

  10. Cómo puedo borrar un tag?

    Con git tag -d [nombre del tag]

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    Con git status

  12. Para que sirve el comando git remote -v?

    Sirve para sirve para mostrar la lista de los repositorios remotos configurados en un repositorio local de Git, mas detallado, incluyendo la URL de cada repositorio remoto.

  13. Cómo le digo a git cual es mi email y correo?

    Con git remote set-url origin git@github.com:/.git

  14. Cuales son los pasos para conectar git con github?

    Con git remote add origin https://github.com/tu-usuario/tu-repositorio.git y git config --list

  15. Cómo puedo crear un alias en git?

    Con git config --global alias.tree "log --graph --decorate --all --oneline"
